# Промпт: каталог «Товары для мастеров»

Используйте этот документ как **единое ТЗ** для наполнения и доработки отдельного каталога материалов для мастеров в Next.js-проекте Salamaha Fine Art (Tailwind 4, `globals.css`, шрифты Cormorant + DM Sans, компоненты `Button`, `ProductCard`, `LandingShell`).

Каталог **не пересекается** с магазином изделий: другой URL, другие slug, другой источник данных.

---

## 1. Назначение и аудитория

- Страница для **мастеров, которые работают с эпоксидной смолой**: материалы, формы, заготовки, вспомогательные инструменты.
- Язык: **русский**. Тон — как на сайте: спокойный, мастерская, без «маркетплейс-шума».
- Заказ через **WhatsApp** (как на карточках основного каталога), без корзины.

---

## 2. Точка входа

- В шапке сайта — зелёная кнопка **«Товары для мастеров»** (`variant="primary"`, как основной CTA).
- Кнопки «Магазин» и «Написать» в шапке **не показываются** (магазин изделий доступен из навигации и подвала).
- Ссылка в подвале: «Товары для мастеров».

**Маршрут каталога:** `/tovary-dlya-masterov`  
**Карточка товара:** `/tovary-dlya-masterov/{slug}`

Основной магазин: `/catalog` и `/catalog/{slug}` — **не смешивать** slug и данные.

---

## 3. Структура каталога (контент)

На странице списка товары логически сгруппированы. В фильтре слева — те же группы. Ниже — **целевая структура** (что должно быть в каталоге).

### 1. Силикагель

Отдельная категория. **Четыре фасовки** — отдельные карточки (или одна карточка с выбором объёма на странице товара; для MVP — четыре SKU):

| slug (пример) | Название в карточке | Объём |
|---------------|---------------------|-------|
| `silikagel-3kg` | Силикагель | 3 кг |
| `silikagel-5kg` | Силикагель | 5 кг |
| `silikagel-10kg` | Силикагель | 10 кг |
| `silikagel-25kg` | Силикагель | 25 кг |

В карточке явно указать объём. В описании: для сушки/абсорбции в мастерской, упаковка, условия хранения (уточнить с мастерской).

### 2. Деревянные заготовки

Категория **«Деревянные заготовки»** — слэбы, плашки, заготовки под заливку (размеры и породы — по факту склада).

Примеры карточек (заполнить реальными позициями):

- Слэб орех, размер …
- Слэб карамельный ясень, размер …
- Заготовка под час / под подставку …

Пока фото нет — заглушка `/product-photo-soon.svg` с текстом «Скоро появятся фото».

### 3. Формы для смолы

Родительская группа в навигации фильтра: **«Формы для смолы»**. Подкатегории:

#### 3.1 Круги

- **С дном** — отдельные карточки по диаметрам (Ø … см) или одна карточка «Круг с дном» с вариантами в WhatsApp.
- **Без дна** — то же для форм без дна.

Фильтр: `formy-krugi-dom`, `formy-krugi-bez-dna`.

#### 3.2 Прямоугольники

- **С дном**
- **Без дна**

Фильтр: `formy-pryam-dom`, `formy-pryam-bez-dna`.

#### 3.3 Набор «круги с дном»

Один или несколько наборов (разный диаметр / количество форм в наборе).

Фильтр: `formy-nabor-krugi-dom`.

#### 3.4 Набор «круги с универсальным дном»

Набор круговых форм с **универсальным дном** (одно дно на несколько диаметров — уточнить в тексте).

Фильтр: `formy-nabor-krugi-universal`.

#### 3.5 Пазловые формы

Формы пазлового типа (органические стыки, мозаика, river table и т.д.).

Фильтр: `formy-pazl`.

### 4. Полезные инструменты

Категория вспомогательных инструментов для мастерской.

**Обязательная позиция (шаблон):**

| slug (пример) | Название |
|---------------|----------|
| `shablon-plashki-osnovy` | Шаблон плашки основы |

Дополнительно (по желанию): уровень, шпатели, мерные ёмкости — **не смешивать** с основным каталогом изделий.

---

## 4. UX страницы списка

- Общий layout: `LandingShell` (шапка, подвал).
- H1: **«Товары для мастеров»**.
- Короткий вводный текст + ссылка на **основной магазин** `/catalog` («изделия и сувениры — в другом каталоге»).
- **Слева:** фильтры (sticky на desktop):
  - группы категорий как в §3;
  - ползунки цены (как в `/catalog`, но query только на `/tovary-dlya-masterov?cat=…`).
- **Справа:** сетка карточек `ProductCard` с `hrefPrefix="/tovary-dlya-masterov"`.
- Пустой результат фильтра — сообщение + «Сбросить фильтры».

Визуал: как основной каталог (кремовые карточки, зелёный текст, aspect 4/5 для фото).

---

## 5. Карточка товара

- H1 — название позиции.
- Фото (или заглушка).
- Цена («от … ₽» или фикс — из данных).
- Текст: материалы для мастеров, уточнение наличия и доставки в WhatsApp.
- CTA: **«Заказать в WhatsApp»** с префиксом в сообщении: `Интересует (товары для мастеров): …`
- Ссылка «Другие товары» → `/tovary-dlya-masterov`.
- Ссылка на основной каталог изделий.

## 7. Медиа

- Путь заглушки: `/product-photo-soon.svg` («Скоро появятся фото»).
- Реальные фото: `public/masters/` с именами по slug, например `public/masters/silikagel-3kg.webp`.
- В `image` указать `/masters/silikagel-3kg.webp`.

Чеклист медиа — дополнить [`docs/SAYT_KARTOCHKI_DLYA_KONTENTA.md`](SAYT_KARTOCHKI_DLYA_KONTENTA.md) отдельным блоком «Товары для мастеров».

## 8. SEO

- `title`: «Товары для мастеров — силикагель, формы, заготовки | Salamaha Fine Art» (согласовать).
- `description`: силикагель, деревянные заготовки, силиконовые формы, инструменты.
- В `sitemap.xml` — `/tovary-dlya-masterov` и все `/tovary-dlya-masterov/{slug}`.
- Не дублировать URL основного каталога.

---

## 9. Что не делать

- Не добавлять эти товары в `ALL_PRODUCTS` / `/catalog`.
- Не использовать те же `slug`, что у изделий (столы, подносы и т.д.).
- Не возвращать квиз «Подбор за 2 минуты» в шапку вместо кнопки каталога мастеров.

---

## 10. Чеклист для исполнителя (ИИ / разработчик)

- [ ] Категории и товары в `masters-products.ts` соответствуют §3.
- [ ] Силикагель: 3 / 5 / 10 / 25 кг — четыре позиции.
- [ ] Формы: все 7 подтипов из §3.1–3.5 представлены в фильтре.
- [ ] «Шаблон плашки основы» в категории инструментов.
- [ ] Фильтр `?cat=` работает только на `/tovary-dlya-masterov`.
- [ ] Карточки ведут на `/tovary-dlya-masterov/{slug}`.
- [ ] Заглушки фото до появления снимков.
- [ ] Цены и тексты согласованы с мастерской (сейчас — ориентировочные «от … ₽»).

---

## 11. Промпт для копирайта карточек (короткий)

> Напиши названия и короткие описания для каталога «Товары для мастеров» студии Salamaha Fine Art: силикагель 3/5/10/25 кг; деревянные заготовки; силиконовые формы (круги и прямоугольники с дном и без, наборы кругов, пазловые); шаблон плашки основы. Стиль — спокойный, для практикующих мастеров, без канцелярита. Цены оставить как «уточняйте в WhatsApp» или «от … ₽».
